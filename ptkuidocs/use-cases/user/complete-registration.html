Endpoint <api-domain>/ptk/newUser/<registration_id>
Method POST
Content-type JSON

Action on response

HRC Response Type Action
200 10:SUCCESS_REGISTRATION_COMPLETE text
  • Remove the form from the page.
  • Show message "Registration complete".
422 3x:ERROR_INVALID_[INVALID_FIELD_NAME] string
  • For error codes, see this document.
  • Error messages remain same as given above in their description.
  • The error code 34 indicates there is some problem with UI.

  • The error code 36 may be returned for <registration_id> in case when <registration_id> had expired by the time user submitted the form or <registration_id> does not exist in the system. The returned field name will be registrationId.

    Code Error Message
    registrationId
    36 Registration was unsuccessful due to expiration of registration id.
500 41:ERROR_SYSTEM_INACCESSIBLE string Pass response to error handler.
500 41:ERROR_UNKNOWN string Pass response to error handler.
